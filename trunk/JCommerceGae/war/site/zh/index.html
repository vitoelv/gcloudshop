最新进展

2010年3月5日， gcShop 0.8.0 build 6 上线。添加了可下载的安装包。添加了apache-ant脚本。为公网门户改进了国际化支持。添加了英文资源文件。升级至Google AppEngine Java 1.3.1

2010年2月3日, gcShop 0.8.0 build 1 上线。添加了订单编辑和订单合并，以及文章管理功能。并且更新了网站主页的“联系我们”

项目介绍

gcShop是运行在Google云计算平台： Google App Engine上的独立网店程序。

“gcShop”的含义为： “google cloud shop” 意为“谷歌云商店”。

gcShop设计目标是一个与google的云计算平台和google的软件和服务整合的，基于Java的，易于扩展和定制的独立网店系统。

gcShop是免费的，以LGPLV3协议发布。
